# codeforces-backup<br>
A desktop Java Application build using Jsoup API that is used to backup codeforces AC submissions of any user.<br>
NOTE: Gym problems will not be backed up because they require login for that particular user.<br>
This tool will work only in windows.<br>
Download Link : https://goo.gl/jssu9M<br>
Download the file and extract it anywhere and then run autorun.exe or autorun.rar.
